# KnowledgeGraph
Crawler_poetry.py ——诗词网网络爬虫爬取唐，宋，古诗词各300篇的脚本
build_graph.py ——建立知识图谱
conversation_graph.py ——问题过滤脚本
question_parser.py ——转化为sql查询语句
answer_search.py ——寻找最终的答案
poem_chat.py ——集成上述功能的问答系统
